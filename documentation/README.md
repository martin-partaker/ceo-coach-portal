# CEO Coach Portal — Documentation

The CEO Coach Portal is the internal tool the coaching team uses to turn a CEO's monthly inputs (weekly journals, session transcripts, KPIs) into a polished **Monthly Progress Summary** — reviewed and edited by the coach, then downloaded as a CEO-facing PDF.

## The 60-second overview

```
Weekly journals (Tally) ─┐
Session transcripts (Zoom)├─► Ingestion ─► Triage ─► matched to a CEO + monthly cycle
KPIs / goals (in-app) ────┘                                   │
                                                              ▼
                                         AI report pipeline (5 stages, Claude)
                                                              │
                                       Coach reviews + edits each section in-app
                                                              │
                                                       Download CEO PDF
```

- **CEO** = a coaching client. **Team** = 2+ CEOs coached together (one shared 10x goal, one joint report). **Cycle** = one month.
- Incoming data is matched to a CEO **by email**; anything unmatched waits in **Triage** for a human.
- Reports are generated by a five-stage AI pipeline (extract facts → find patterns → draft → critique → refine), then rendered to PDF.

## Tech stack (at a glance)
Next.js 16 (App Router) · tRPC v11 · Drizzle ORM · Neon Postgres (eu-west-2) · Neon Auth · Vercel · **Anthropic Claude** (report generation) · Zoom (transcripts) · Tally (forms). See [ARCHITECTURE.md](./ARCHITECTURE.md) for detail.
