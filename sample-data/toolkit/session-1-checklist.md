# Session 1 checklist (synthetic)

## Success criteria
Client leaves **focused + energized**, with:
- a believable (but stretching) 10X destination
- the #1 constraint named
- 30-day commitments agreed

## Before the call (5 minutes)
- Read intake + last cycle notes (if any)
- Identify:
  - what’s unclear about the 10X destination
  - likely constraints (product / market / delivery / people)
- Decide 2–3 “devil’s advocate” questions

## Call flow (simple)
1. Rapport + frame the program (2–3 mins)
2. Destination: state the 10X goal in one sentence; test believability (15 mins)
3. Business model snapshot: product / market / delivery (10 mins)
4. Constraint: pick the single highest-leverage constraint (10 mins)
5. Commitments: 30-day priorities + weekly momentum journal cadence (10 mins)
6. Close: client repeats commitments; schedule next call (2 mins)

## Red flags to watch
- Vague goal (“grow a lot”) or externally dependent goal with no controllable levers
- Client avoiding the uncomfortable conversation / hire / decision
- No calendar time allocated for momentum journal
